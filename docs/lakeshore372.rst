=============
Lakeshore 372
=============

The Lakeshore 372 (LS372) units are used for 100 mK and 1K thermometer readout. Basic functionality to interface and control an LS372 is provided by the ``ocs.Lakeshore.Lakeshore372.py`` module and its supplimentary modules (i.e. ``ocs.Lakeshore.channel372.py``).
